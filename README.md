# AI Music Generator

## Description

The AI Music Generator is a web-based application that automatically generates random musical melodies and plays them in the browser. It demonstrates basic AI-inspired music creation using JavaScript and the Web Audio API.

## Features

* Automatic melody generation
* Real-time audio playback
* Interactive user interface
* Random note selection
* Browser-based music generation

## Technologies Used

* HTML
* CSS
* JavaScript
* Web Audio API

## How to Run

1. Download the project files.
2. Open the project folder in VS Code.
3. Open index.html in a web browser.
4. Click the Generate Music button.
5. Listen to the generated melody.
